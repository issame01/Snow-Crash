Step 3 вАУ What About the Token?

Now test:

./level09 $(cat token)

The output is still unreadable.

That means:

The token itself was encoded using this algorithm.

To recover the original password, we must reverse it.

Reverse the Logic

If:

encoded[i] = original[i] + i

Then:

original[i] = encoded[i] - i

We must subtract the index from each character.


Step 4 вАУ Write a Decoder Program

Create a small C program:

int main(int argc, char *argv[])
{
    int i = 0;
    if (argc != 2)
        return (1);

    while (argv[1][i])
    {
        printf("%c", argv[1][i] - i);
        i++;
    }
    return 0;
}

Compile it:

Now run:
